These are all of the icons which you can use within the projects of Provincie Zuid-Holland.

## Usage

[![npm version](https://img.shields.io/npm/v/@pzh-ui/icons.svg?label=@pzh-ui%2Ficons)](https://npmjs.com/package/@pzh-ui/icons)

`yarn add @pzh-ui/icons`

```jsx
import { Check } from '@pzh-ui/icons'

function Page() {
    return <Check size={20} />
}
```
