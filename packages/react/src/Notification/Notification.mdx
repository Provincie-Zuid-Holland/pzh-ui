# Notification

Een component voor het tonen van contextuele meldingen, zoals informatie, waarschuwingen, bevestigingen en foutmeldingen.

## Gebruik

```tsx
import { Notification } from '@pzh-ui/react'

export const Example = () => (
    <Notification variant="info" title="Titel van melding">
        Dit is een toelichting.
    </Notification>
)
```

## Accessibility

- De melding gebruikt tekstuele content voor duidelijke communicatie.
- Kleur wordt gecombineerd met een icoon, zodat betekenis niet alleen afhankelijk is van kleur.
- Gebruik bij dynamische meldingen eventueel aanvullende ARIA-patterns, zoals `role="status"` of `role="alert"`, afhankelijk van de context.
